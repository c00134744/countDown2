## 需求
开发一款安卓APP，用于倒计时。 主要特性有：
1. 最大支持45min倒计时
2. 界面简洁，示意图参考 countdownUI.png
3. 计时结束时提醒用户，提醒方式时响铃+震动。 提醒时间默认时0.5s，不会一直处于提醒状态。

## 界面详细特性
1. 计时点按钮
    - 既用于配置倒计时的时长，也用于指示当前计时的进程
    - 位于计时圈轨道上，类似于卫星轨道上的卫星
    - 拖动可以设置计时时长，拉到左侧下方，支持最大45min倒计时
    - 在倒计时过程中，此按钮沿着圆形轨道，顺时钟移动
    - 圆形轨道左下角是起点，右下角是终点

2. 计时圈
    - 类似于行星上的卫星轨道
    - 控制按钮走过的轨道（也就是计时经过的范围），显示成灰色。未走过的轨道，显示成绿色

3. 用户控制按钮
    - 支持倒计时的开始/暂停/停止
    - 用户配置好计时时长后，点击就开始进入倒计时
    - 倒计时过程中，点击暂停，再次点击继续计时
    - 倒计时过程中，长按停止计时，剩余时间清零

4. 倒计时时间
    - 位于倒计时圈内部中间位置，暂时当前倒计时的剩余时间

5. 低功耗特性
    - 倒计时期间，保持屏幕常亮
    - 计时结束5s后，系统可以灭屏休眠
    - 倒计时过程中，如果应用被切到后台，也要能够继续计时和提醒

5. 界面风格
    - 参考Glassmorphism风格对UI进行优化
    - 暗黑色系
